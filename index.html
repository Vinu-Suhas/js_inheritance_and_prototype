        Q2.Write code to explain prototype chaining

        function Animal(name){
            this.name=name
        }
        
        
        Animal.prototype.breed="British Shorthair"
        Animal.prototype.info=function(){
            return `This ${this.name} of breed ${this.breed}`
        }
        
        animalobj=new Animal('cat')
        console.log(animalobj)
        console.log(animalobj.breed)
        console.log(animalobj.info())


        Explaination :
        
        The syntax of writing prototype for a function is 
            function_name.prototype.new_variable=value_or _function _to_be_assigned 
        
           where prototype is a keyword  and this value or function can be accessed in any Object of that  function after the declartion of the prototype
